# Shreddit - Workout Tracker

Mobile-first PWA for tracking your bodybuilding program.

## Setup

```bash
npm install
npm run dev
```

Open on your phone by visiting your computer's local IP on port 3000.

## Features

- Week/day navigation
- Exercise tracking (weight & reps)
- Local storage persistence
- Mobile-optimized UI
- PWA installable to home screen
